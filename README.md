# Dynamixel-12A-ESPBlynk

Hardware

1. NodeMCU ESP8266
2. Dynamixel Ax-12A Robot Actuator
3. Jumper wires
4. power supply dc 12V
5. mobile phone with Blynk app

Arduino library to control Dynamixel AX-12A <br>
https://github.com/ThingType/AX-12A-servo-library
